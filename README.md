------

#### Installation :

`git clone https://github.com/TshiSelle/eirene.git`

------

#### Setup :

* **Server** :

  `cd .\backend`

  `node server_connect.js`

  or

  `nodemon server_connect.js`

- **Client** :

  `npm start`

------

#### .env File Format :

```

PORT = 8080

CLIENT_PORT = 3000

MONGO_CONNECTION_URL = 'yourmongodburi'

JWT_SECRET = 'securekey'

LOOM_EMAIL_PASSWORD = 'emailpass'

```
